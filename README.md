# Side Quest Week 6

## Setup and Interaction Instructions

To run the sketch locally, open `index.html` in Google Chrome using Live Server.

**Controls:**
- Move: WASD
- Shoot: Spacebar (shoots in the direction you last moved)
- B: Skip to boss fight (testing only)
- Restart: R (after win or game over)

As a werewolf, explore the world, avoid the lava moon blocks and survive enemy waves of villagers as you move north to avoid getting voted out from the dimension. Then, enter the glowing boss zone to fight the giant orange blob. Watch the minimap to track enemies off screen.
